# Sobre o projeto
O projeto foi criado para ajudar um amigo a organizar projetos de audiovisual, criando assim templates para as pastas e um .txt informando sobre o que é o projeto.

# Como utilizar
É um programa simples, basta rodá-lo diretamente no terminal e preencher os dados pedidos para a criação do projeto.  

Caso haja a necessidade de criar um .exe deste programa, é só rodar os seguintes comandos:  

- Baixar Biblioteca:

```terminal
pip install pyinstaller
```

- Criar o *.exe*:

```terminal
pyinstaller --onefile Criador_de_projetos.py
```
